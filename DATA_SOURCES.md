# Data Sources

## Twitter Sentiment Dataset
- **Source:** X/Twitter posts collected via the Twitter API
- **Files:**
  - `twitter_training.csv` — 74,682 labeled samples
  - `twitter_validation.csv` — 1,511 labeled samples
- **Labels:** Positive, Negative, Neutral, Irrelevant
- **License:** Publicly available for research and educational use
- **Columns:** ID, Category (game/subject), Sentiment (label), Text
